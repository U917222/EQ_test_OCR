# Archived Google Apps Script implementation

This directory contains the retired Google Apps Script application and its OCR callback adapter.
It is kept only as a historical reference and is not part of the active build, test, or deployment path.

Contents:

- `apps-script/`: former Apps Script backend and web UI
- `ocr-api/`: former Drive download and webhook callback helpers
- `sync-gas.sh`: former scoring-core copy script
- `gasRank.test.js`: former regression test against Apps Script source

Do not add new features here. Current development belongs in `web/`, `scoring-api/`, `ocr-api/`, and `scoring-core/`.
